## Описание проекта
GitHub User Finder - это desktop-приложение с графическим интерфейсом для поиска пользователей GitHub, просмотра их информации и сохранения избранных пользователей в локальное хранилище.

### Основные возможности:
- **Поиск пользователей** GitHub по имени
- **Просмотр информации** о пользователях (репозитории, подписчики)
- **Добавление в избранное** понравившихся пользователей
- **Сохранение избранного** в JSON файл
- **Экспорт избранных** пользователей
- **Контекстное меню** для удобного управления

## Используемые технологии
- **Python 3.8+**
- **Tkinter** - для графического интерфейса
- **Requests** - для работы с GitHub API
- **JSON** - для хранения данных

## Как использовать GitHub API

### Получение токена доступа (опционально)
Для увеличения лимита запросов рекомендуется получить персональный токен:

1. Перейдите на GitHub → Settings → Developer settings → Personal access tokens
2. Нажмите "Generate new token (classic)"
3. Выберите права (достаточно `public_repo` и `user`)
4. Скопируйте сгенерированный токен

##Примеры использования
**Сценарий 1. Поиск пользователя**
Действия:

- Введите в поле поиска: octocat.

- Нажмите «Найти».

**Ожидаемый результат:**

- В таблице результатов появляется пользователь octocat с информацией.

- История поиска обновляется.

### Сценарий 2. Добавление в избранные
**Действия:**

- Найдите пользователя (например, torvalds).

- Нажмите «В избранные» рядом с его именем.

**Ожидаемый результат:**

- Пользователь появляется в таблице «Избранные».

- Появляется сообщение: «Пользователь добавлен в избранные!».

### Сценарий 3. Открытие профиля
**Действия:**

- Выберите любого пользователя в результатах поиска.

- Нажмите «Открыть профиль».

**Ожидаемый результат:**

- Браузер открывается на странице пользователя GitHub.

### Сценарий 4. Сохранение и загрузка избранных
**Действия:**

- Добавьте несколько пользователей в избранные.

- Нажмите «Сохранить избранных».

- Закройте и снова откройте приложение.

- Нажмите «Загрузить избранных».

**Ожидаемый результат:**

- Восстанавливается список избранных пользователей.
