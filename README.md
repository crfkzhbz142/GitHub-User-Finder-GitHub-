## Описание
GitHub User Finder - это desktop-приложение с графическим интерфейсом для поиска пользователей GitHub, просмотра их информации и сохранения избранных пользователей в локальное хранилище.

### Основные возможности:
- **Поиск пользователей** GitHub по имени
- **Просмотр информации** о пользователях (репозитории, подписчики)
- **Добавление в избранное** понравившихся пользователей
- **Сохранение избранного** в JSON файл
- **Экспорт избранных** пользователей
- **Контекстное меню** для удобного управления

## Технологии
- **Python 3.8+**
- **Tkinter** - для графического интерфейса
- **Requests** - для работы с GitHub API
- **JSON** - для хранения данных

## Как использовать GitHub API

### Получение токена доступа (опционально)
Для увеличения лимита запросов рекомендуется получить персональный токен:

1. Перейдите на GitHub → Settings → Developer settings → Personal access tokens
2. Нажмите "Generate new token (classic)"
3. Выберите права (достаточно `public_repo` и `user`)
4. Скопируйте сгенерированный токен
